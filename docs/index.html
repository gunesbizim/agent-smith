▸ What Agent Smith Does

🔍 Analyze

Scans your repo to detect exact frameworks, ORMs, auth libraries, validation, logging, DB drivers, cache, test frameworks, linters — with version numbers. Reads lock files from npm, pnpm, Go, Python, PHP, Rust.

📋 Interview

Asks 9 questions about your team's conventions before generating anything. Branch naming, commit format, PR checklist, testing requirements. Smart defaults based on detection. Type ? for elaboration.

🧬 Scaffold

Generates 12+ skill files customized to YOUR stack. Pre-push gates use your actual tools. Commands reference your packages with versions. No Django defaults leaking into your Go project.

⚙ Configure

Installs and configures 11 MCP servers. Writes .claude/settings.json, .mcp.json, architecture docs. Sets up automation hooks. Restart Claude Code and everything works.

▸ Automation Hooks

🩺 SessionStart

Runs health check at every session start. Reports missing MCPs, uncommitted changes, stale gitnexus index, setup status.

🛡 PreToolUse

Intercepts git commands — validates conventional commit format, shows pre-push gates, warns on destructive operations. Suspends caveman during memory writes.

📋 Stop

Detects uncommitted changes at session end. Categorizes by backend/frontend/docs. Suggests /git and /documentation.

🧠 PostToolUse

Resumes caveman after memory writes. Edge case: never auto-enables caveman if user explicitly turned it off.

Quick Start

# Install globally
npm i -g @gunesbizim/agent-smith

# Bootstrap a project
cd your-project
agent-smith init

# Or one-shot via npx
npx @gunesbizim/agent-smith init --dir /path/to/project

Claude Code Plugin

# Install as a Claude Code plugin
claude plugin install @gunesbizim/agent-smith

# Or from GitHub
claude plugin install github:gunesbizim/agent-smith
